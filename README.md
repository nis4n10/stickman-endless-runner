**Stickman Endless Runner** is an interactive **Python game** built using **Pygame**, where the player controls a stickman running endlessly through a scrolling environment.  

The goal is to **avoid obstacles** and survive as long as possible to achieve a high score. This project combines **custom stickman animation**, **interactive buttons**, and **endless gameplay**, making it **fun, visually appealing, and teacher-approved**.

---

## ðŸŽ® Features

- **Stickman Character:** Fully drawn using lines and circles, no sprites.  
- **Jump Mechanic:** Press `SPACEBAR` to jump over obstacles.  
- **Random Obstacles:** Endless obstacles scroll from right to left.  
- **Score Tracking:** Score increases based on survival time.  
- **Game Over Screen:** Clickable **Restart** and **Quit** buttons.  
- **Interactive Buttons:** Start, Restart, and Quit buttons with hover effects.  
- **Teacher-Friendly:** No cars or road graphics.
